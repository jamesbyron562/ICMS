# ICMS(Internet Cafe Management System)

First progress: Login Form Design & Toolbox Setup 
Members Involved: Whole Group

In the first stage, we focused on creating the Login Form.
We used the toolbox in Windows Forms to add essential controls such as:

Labels (for username and password)
Textboxes (for user input)
Button (for login action)

We also designed the layout by arranging the controls properly and improving the appearance (alignment, spacing, and basic styling).
At this stage, the goal was to build a clean and functional login interface.

May 1, 2026 - Dashboard Form Development 
Members Involved: Whole Group

After completing the login form, we proceeded to create the Dashboard Form, which appears after a successful login.
In this stage:

We created a new form for the dashboard
Connected it to the login form (navigation after login)
Designed the layout for the main system interface

The dashboard serves as the main menu or central area of the system where users can access other features.

Members Involved: Whole Group

After finishing the dashboard form, we started creating the other forms needed in the system such as the Transaction Form, Profile Form, Account Form, and other related forms. Each form was designed with its own controls and functions depending on its purpose in the system.

During this stage, we encountered difficulties in linking all the forms together, especially in navigating between forms properly. Some methods did not work as expected, so the group tried different approaches until we found an alternative way to make the forms connected and functional.
